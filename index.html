Konjac by mootoh

Input Method using Translate APIs.

Konjac is an Input Method using translate API. FAQ --- * What Konjac is? * Read [this story](http://doraemon.mangawiki.org/translation-konjac/).

Install

Copy Konjac.app to /Library/Input Methods.
